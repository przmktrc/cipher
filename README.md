# CIPHER


## Overwiev
This is cipher, a set of programs for ciphering & deciphering stuff using morse code (not yet implemented), Caesar cipher (NYI), exchange ciphers (NYI) and more (NYI)!


## Note
Note: if your cmake compains about wrong version, feel free to change it; I haven't tested any other cmake versions and the 3.22 is just the one I use. If you find an earlier version working, please submit a PR.


## Requirements
- cmake
- C++20
- [{fmt}](https://github.com/fmtlib/fmt); though you may have to change the [CMakeLists.txt](CMakeLists.txt) to get it working
